## 🚀 HEC Generative AI Training - Cohort 3

This repository contains all my work, assignments, and learning materials from the **HEC Generative AI Training Program - Cohort 3**. The program focuses on building strong foundations in Artificial Intelligence, Machine Learning, and Generative AI technologies.

---

## 📌 Objectives

- Understand core concepts of AI and Machine Learning  
- Explore Generative AI models and their real-world applications  
- Gain hands-on experience with Python and AI tools  
- Build practical projects to strengthen problem-solving skills  

---

## 🧠 Topics Covered

- Introduction to Artificial Intelligence  
- Machine Learning Basics  
- Deep Learning Fundamentals  
- Generative AI Concepts  
- Natural Language Processing (NLP)  
- Prompt Engineering  
- AI Tools & Frameworks
