# 🌦 Weather App using Python & Gradio

## 📌 Overview
This project is a simple interactive Weather Application built using Python. It allows users to enter a city name and fetch real-time weather information using an API.

## 🚀 Features
- 🌍 Search weather by city name  
- 🌡 Real-time temperature (°C)  
- 💧 Humidity details  
- 🌥 Weather condition description  
- ❌ Error handling for invalid cities  
- 🎨 Clean and interactive UI using Gradio  

## 🛠 Tech Stack
- Python  
- Gradio  
- Requests library  
- OpenWeatherMap API  

## ⚙️ How it Works
1. User enters a city name  
2. API fetches live weather data  
3. Data is processed in Python  
4. Results are displayed in a Gradio interface
